Графики

Визуализация фактических и прогнозных значений временного ряда.

Файл может содержать числа через новую строку, точку с запятой, табуляцию или запятую. Для дробей с десятичной запятой лучше использовать ; или перенос строки.
Методы прогнозирования:
• average — константа по среднему значению
• moving — константа по последним значениям
• holt — Holt / экспоненциальное сглаживание с трендом
• neural — нейросетевой прогноз MLP
